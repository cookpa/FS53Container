# FS53Container

Docker container for CentOS 6 and FreeSurfer 5.3.

## Other FreeSurfer containers

In addition to the [official FreeSurfer docker images](https://hub.docker.com/u/freesurfer), which date back to FS 6.0.0, developer pwighton has some more images including 5.3-HCP. See [here](https://hub.docker.com/r/pwighton/freesurfer/tags) for available tags.
